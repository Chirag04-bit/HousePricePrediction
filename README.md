# House Price Prediction

This project focuses on predicting house prices using Machine Learning techniques. The housing dataset was cleaned, analyzed, and preprocessed before training Linear Regression and Random Forest Regression models.

## Features
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Correlation Analysis
- Linear Regression Model
- Random Forest Regressor
- Model Evaluation (MAE, RMSE, R² Score)
- Price Distribution Visualization
- Correlation Heatmap
- Actual vs Predicted Price Analysis

## Results
- Linear Regression achieved an R² score of approximately 0.65.
- Area and number of bathrooms were the strongest predictors of house prices.
- Air conditioning, parking, and number of stories also showed significant influence on property value.
- The model successfully captures major trends in housing prices and provides useful insights for real estate decision-making.
